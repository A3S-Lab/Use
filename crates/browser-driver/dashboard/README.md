# A3S Use Browser Dashboard

This static Next.js dashboard is derived from the `packages/dashboard`
application in `vercel-labs/agent-browser` 0.32.1 at commit
`2b202640ee89dc7aadb5e8c9d600e089e9056985` and is distributed under the
Apache-2.0 license recorded in `../UPSTREAM.md`.

Build the checked-in static bundle with:

```text
pnpm install --ignore-workspace
pnpm build
```
